Event Studio — Rethinking Event Setup

Redesigning event configuration for enterprise platforms to reduce organizer friction.

In Progress UX Rapid Prototyping Claude Code

Role: UX design, rapid prototyping

The Problem

The complexity of configuring events in enterprise platforms creates friction that slows organizer workflows. Setting up a new event involves multiple steps across different screens, with limited guidance on required fields and dependencies between settings.

What I'm Doing

I used customer feedback to drive UX and design choices, rapidly prototyped a redesigned event setup experience, and wrote an implementation spec using Claude Code. I then handed both over to leadership to be incorporated into the larger project.

This case study will be published when the feature ships.
